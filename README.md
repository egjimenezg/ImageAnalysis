ImageAnalysis
=============

Proyecto de Análisis de Imágenes

**Operaciones puntuales**
- Negativo de una imágen
- Conversión a nivel de gris
- Conversión a modelo de color (Opcional)
- Operaciones Aritméticas
- Operaciones Lógicas
- Componentes conexas
- Umbralización

**Mejoramiento de la imagen**
- Histograma
- Ajuste de Brillo
- Ecualización
- Estrechamiento
- Expansión


**Filtros**
- De orden
 - Media
 - Moda
 - Mediana
 - Máximo
 - Mínimo

- Lineales
 - Pasa altas
 - Pasa bajas
